## 📖 About

This repository contains the solution to **Assignment 8** of the Object-Oriented Programming course.
This assignment focuses on **Templates (Generic Programming)** in C++, allowing functions and classes to work with different data types without rewriting code.

It demonstrates both **function templates** and **class templates** with practical examples like sorting, searching, stack, queue, and calculator implementations.

## 🧠 Concepts Used

* **Function Templates** — write one function for multiple data types
* **Class Templates** — create generic classes
* **Generic Programming** — code reuse and flexibility
* **Data Structures using Templates**

  * Stack (LIFO)
  * Queue (FIFO)
* **Searching & Sorting Algorithms**

  * Linear Search
  * Bubble Sort
* **Template with Multiple Types** — using multiple template parameters

## ▶️ How to Run

1. **Clone this repository:**

   ```bash
   git clone https://github.com/parminderrathour/OOPS-assignment-8.git
   cd OOPS-assignment-8
   ```

2. **Compile the file using g++:**

   ```bash
   g++ parminder_ass8.cpp -o output
   ```

3. **Run the program:**

   ```bash
   ./output
   ```

> 💡 Only one question’s `main()` is active at a time.
> To run another question, comment/uncomment the respective code block.
